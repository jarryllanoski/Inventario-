# 📦 Auditoría de Inventario

App web para auditar inventario escaneando códigos de barras con la cámara del celular.

## 🚀 Cómo usar

Abre la app en: `https://TU_USUARIO.github.io/inventario`

## ✅ Funciones

- Sube tu Excel con la lista de inventario
- Detecta columnas automáticamente
- Escanea códigos EAN-13, EAN-8, Code 128, Code 39
- Marca productos auditados por gandola/isla
- Exporta reporte de faltantes
